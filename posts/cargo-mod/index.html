Rust 本地模块调用示例

目录结构如下

.
├── bar
│   ├── Cargo.lock
│   ├── Cargo.toml
│   └── src
│       ├── hello
│       │   ├── greet.rs
│       │   └── mod.rs
│       └── lib.rs
└── foo
    ├── Cargo.lock
    ├── Cargo.toml
    └── src
        └── main.rs

bar 工程 ( Library Project )

  • greet.rs
pub fn greeting() {
    println!("hello world");
}
  • mod.rs
pub mod greet;
  • lib.rs
pub mod hello;

foo 工程 ( Binary Project )

  • Cargo.toml
[package]
name = "foo"
version = "0.1.0"
authors = ["wimi <ywm00@qq.com>"]
edition = "2018"

[dependencies]
bar = { path = "../bar" }
  • main.rs
extern crate bar;

use bar::hello::greet;

fn main() {
    greet::greeting();
}
