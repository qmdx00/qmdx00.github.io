本地不同项目之间模块调用示例

目录结构如下

.
├── bar
│   ├── Cargo.lock
│   ├── Cargo.toml
│   └── src
│       ├── hello
│       │   ├── greet.rs
│       │   └── mod.rs
│       └── lib.rs
└── foo
    ├── Cargo.lock
    ├── Cargo.toml
    └── src
        └── main.rs

bar 工程 ( Library Project )

  • greet.rs
pub fn greeting() {
    println!("hello world");
}
  • mod.rs
pub mod greet;
  • lib.rs
pub mod hello;

foo 工程 ( Binary Project )

  • Cargo.toml
[package]
name = "foo"
version = "0.1.0"
authors = ["wimi <ywm00@qq.com>"]
edition = "2018"

[dependencies]
bar = { path = "../bar" }
  • main.rs
extern crate bar;

use bar::hello::greet;

fn main() {
    greet::greeting();
}

同一项目下模块调用示例

foo/
├── Cargo.lock
├── Cargo.toml
└── src
    ├── bar
    │   ├── hello.rs
    │   └── mod.rs
    ├── lib.rs
    └── main.rs
  • hello.rs
pub fn hello_world() {
    println!("hello world");
}
  • mod.rs
pub mod hello;
  • lib.rs
pub mod bar;
  • main.rs
use foo::bar::hello; // 此处使用本项目的 package name 开始

fn main() {
    hello::hello_world();
}
  • Cargo.toml
[package]
name = "foo"
version = "0.1.0"
authors = ["wimi <ywm00@qq.com>"]
edition = "2018"

[dependencies]
